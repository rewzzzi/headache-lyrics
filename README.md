# Headache - Asal (Lyrics Experience)

Стильный веб-плеер с динамическим фоном и автоматической синхронизацией текста песен. Проект вдохновлен современными музыкальными интерфейсами и плавно отображает строки по мере воспроизведения трека.

## Особенности
* **Синхронизация текста:** Текст автоматически прокручивается и подсвечивается в такт музыке.
* **Интерактивность:** Кликни на любую строчку текста, чтобы мгновенно перемотать трек на этот момент.
* **Кастомный плеер:** Минималистичные элементы управления (Play/Pause, прогресс-бар, громкость).
* **Живой фон:** Анимированный градиент и эффект мягкого размытия (mask-image) для текста.
* **Адаптивность:** Интерфейс корректно отображается на разных экранах.

## Технологии
* **HTML5** (Audio API)
* **CSS3** (Animations, Flexbox, Glassmorphism)
* **JavaScript** (Vanilla JS)
